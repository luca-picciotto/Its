-------------------------------
	RESPONSIVE DESIGN		
-------------------------------
What is?
	- Si aggiunsta in automatico tra i device
	- Si usa nelle pagine web
	- Con le media query cambia il css e il layout
		{
			media-query: 
				istruzioni css che permettono di 
				cambiare gli stili in base alle 
				carratteristicedel dispositivo.
		}
Designers?
	- usano il responsive per partire dal desktop
	- oggi il responsive si usa anche per il mobile-first
	- maggiore scalabilità mobile -> desktop
Developers?
	- il responsive era usato per partire dal desktop-first
	- lo sviluppo mobile è più difficile
